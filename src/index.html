Информация о C

Язык C родился в 1972 году в стенах легендарной исследовательской лаборатории Bell Labs. Его автором стал американский компьютерный ученый Деннис Ритчи. Проект вырос из необходимости переписать операционную систему Unix, которую его коллега Кен Томпсон до этого создавал на ассемблере и промежуточном языке B. Название «C» стало развитием алфавитной логики вслед за «B».

Главной целью Ритчи было создание гибкого, но лаконичного языка, способного заменить ассемблер при системном программировании. Инструмент должен был обеспечивать прямой доступ к аппаратной части компьютера и памяти, но оставаться переносимым. Это позволило бы один раз написать код операционной системы и легко запускать его на совершенно разных компьютерных архитектурах без полной переработки.

Главные черты C — экстремальная скорость выполнения, минимальное потребление ресурсов и скромный объем встроенной библиотеки. Язык предоставляет программисту полную свободу через механизм указателей и ручное управление памятью. В C нет сложного встроенного рантайма или автоматической сборки мусора. Такая простота делает код прозрачным, предсказуемым и невероятно близким к возможностям железа.

C является исключительно компилируемым языком. Компилятор берет исходный текстовый файл и превращает его напрямую в бинарные машинные инструкции процессора. Самыми известными компиляторами сегодня выступают GCC (проект GNU) и современный Clang (на базе LLVM). Эти утилиты не просто переводят код, но и глубоко оптимизируют его под конкретную архитектуру процессора ради максимального быстродействия.

Развитие C жестко регулируется стандартами. Первая официальная спецификация появилась в 1989 году (стандарт ANSI C или C89), зафиксировав базовый синтаксис. Затем вышло крупное обновление C99, добавившее новые типы данных и комментарии //. Современную эпоху представляют стандарты C11 и C17, а новейшая версия C23 привнесла в язык долгожданные улучшения безопасности и макросов.

Первая программа на C

Классическим началом изучения языка является написание программы, которая выводит на экран приветствие. В Си минимальный рабочий код выглядит следующим образом:

C
#include <stdio.h>
    
int main() {
    // Вывод строки на экран
    printf("Hello, World!\n"); 
    return 0;
}

Разбор базовых элементов:

  • #include <stdio.h> — директива препроцессора, которая подключает заголовочный файл стандартной библиотеки ввода-вывода. Без нее компилятор не узнает функцию printf.
  • int main() — главная функция программы. Выполнение любого приложения на C начинается строго с нее.
  • \n — специальный управляющий символ, который переводит курсор на новую строку.
  • return 0; — завершает работу функции main() и возвращает операционной системе сигнал о том, что программа отработала успешно (без ошибок).

Ввод и вывод данных

Для взаимодействия с пользователем в C чаще всего используются функции printf() (для вывода информации на экран) и scanf() (для чтения данных с клавиатуры). Они работают на основе спецификаторов формата.

c
#include <stdio.h>
                    
int main() {
    int userNumber;

    printf("Введите целое число: ");
    // Знак & перед переменной обязателен — он передает адрес памяти
    scanf("%d", &userNumber);

    printf("Вы ввели число: %d\n", userNumber);
    return 0;
}

Важные правила при работе с вводом/выводом:

  • При использовании scanf перед именем переменной ставится оператор взятия адреса & (амперсанд). Он указывает функции, куда именно в памяти компьютера нужно сохранить введенное значение.
  • Спецификатор %d сообщает компилятору, что мы работаем именно с целым десятичным числом (Integer).

Переменные и типы данных

C — это язык со строгой статической типизацией. Это значит, что каждая переменная обязана иметь четко определенный тип, который нельзя изменить в процессе работы программы.

c
#include <stdio.h>

int main() {
    int count = 10;            // Целое число (обычно 4 байта)
    float price = 49.99;       // Число с плавающей точкой (4 байта)
    double pi = 3.1415926535;  // Число с двойной точностью (8 байт)
    char letter = 'F';         // Один символ в одинарных кавычках (1 байт)

    // Вывод разных типов данных с использованием их спецификаторов
    printf("Количество: %d\n", count);
    printf("Цена: %.2f\n", price); // .2 ограничивает вывод до 2 знаков после запятой
    printf("Символ: %c\n", letter);

    return 0;
}

Основные спецификаторы формата (Format Specifiers):

  • %d или %i — для целых чисел (int)
  • %f — для дробных чисел (float)
  • %lf — для дробных чисел повышенной точности (double)
  • %c — для одиночных символов (char)

Операции в языке C

В языке Си доступен богатый набор операторов для выполнения математических, логических и сравнительных действий над переменными.

c
#include <stdio.h>
                    
int main() {
    int a = 10;
    int b = 3;
                        
    // 1. Арифметические операции
    int sum = a + b;       // Сложение (13)
    int diff = a - b;      // Вычитание (7)
    int prod = a * b;      // Умножение (30)
    int quotient = a / b;  // Целочисленное деление (значение: 3)
    int remainder = a % b; // Остаток от деления (значение: 1)

    // 2. Инкремент и декремент
    a++; // Увеличение a на 1 (станет 11)
    b--; // Уменьшение b на 1 (станет 2)

    printf("Деление: %d, Остаток: %d\n", quotient, remainder);
    return 0;
}

Особенности базовых операций:

  • Целочисленное деление: Если вы делите `int` на `int` (например, `10 / 3`), Си отбрасывает дробную часть, и результатом будет `3`, а не `3.333...`. Чтобы получить точный результат, хотя бы одно из чисел должно быть типа `float` или `double`.
  • Взятие остатка (%): Оператор деления по модулю работает только с целыми числами и возвращает остаток, оставшийся после деления одного числа на другое.
